Code Hike is a remark plugin for [MDX v2](https://mdxjs.com/). The specific set up will depend on [your stack](#frameworks), it usually involves five steps:

<Collapsable>

**1.** Set up MDX v2

See the [MDX v2 documentation](https://mdxjs.com/docs/getting-started/).

</Collapsable>
<Collapsable>

**2.** Install the Code Hike plugin

<CH.Code showCopyButton>

```bash
npm install @code-hike/mdx
```

</CH.Code>

or

<CH.Code showCopyButton>

```bash
yarn add @code-hike/mdx
```

</CH.Code>

</Collapsable>
<Collapsable>

**3.** Include Code Hike's CSS

Depending on your stack, it could be:

<CH.Code showCopyButton>

```js your.js
import "@code-hike/mdx/dist/index.css"
```

</CH.Code>

</Collapsable>
<Collapsable>

**4.** Add the remark plugin to your MDX configuration

Also depends on your stack. Find the right place to pass `remarkPlugins`.

{/* prettier-ignore */}
```js your.config.js focus=2,5,6,9
const remarkSomething = require("remark-something");
const { remarkCodeHike } = require("@code-hike/mdx");

mdxOptions = {
  remarkPlugins: [
    [remarkCodeHike, { theme: "github-dark", lineNumbers: false }],
    remarkSomething,
    remarkSomethingElse,
  ],
};
```

</Collapsable>
<Collapsable>

**5.** Copy and try one of the [demos](https://codehike.org/#demos) to test the set up.

For example:


</Collapsable>
