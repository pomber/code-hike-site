The fundamental building block of Code Hike is the code block. There are several ways to write code blocks inside markdown. The most common way is to wrap the code using triple backticks:

{/* prettier-ignore */}
````mdx lorem.mdx focus=3:5
## Lorem ipsum

```
print("Hello, world!")
```

dolor **sit** amet
````

### Syntax highlighting

If you specify a language after the <CH.InlineCode>```</CH.InlineCode> Code Hike will apply syntax highlighting to the code.

<SideBySide>

{/* prettier-ignore */}
````mdx lorem.mdx focus=3:5 mark=3[4:9]
### Lorem ipsum

```

````
